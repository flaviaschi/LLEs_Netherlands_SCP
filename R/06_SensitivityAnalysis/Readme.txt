This subfolder (R) contains all the necessary codes for running the sensitivity analysis.

There are 5 R script, which have self-explanatory names.

Sceanrio1 refers to Maximize ES Sceanrio
Sceanrio2 refers to Minimize Costs Scenario 

The subfolders are labeled by number to follow the order of the analysis. 
Each R scripts contains an explanation of what the script is for.
